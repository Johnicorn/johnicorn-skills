# Johnicorn Skills

Johnicorn Agentic Skill katalogs.

Pirmais skils ir `lv-antislop`, latviešu valodas stila un vārdu formu pārbaudei.

## Uzstādīšana

```text
/plugin marketplace add Johnicorn/johnicorn-skills
/plugin install lv-antislop@johnicorn-skills
```

## Skili

| Skils | Apraksts |
| --- | --- |
| `lv-antislop` | Latviešu valodas de-slop skils ar B1/B2/B3 pārbaudes vārtiem. |
